
# Sorting Algorithms Performance Analysis

This project compares the performance of Quick Sort, Heap Sort, and Insertion Sort using Python and a real-world Amazon Reviews dataset.

## Project Overview
The objective of this project is to evaluate the theoretical and practical performance of different sorting algorithms in an e-commerce scenario.

## Algorithms Used
- Quick Sort
- Heap Sort
- Insertion Sort

## Dataset
Amazon Reviews Dataset (Kaggle):
https://www.kaggle.com/datasets/dongrelaxman/amazon-reviews-dataset

The dataset contains over 21,000 customer reviews and was used to analyze the scalability and efficiency of different sorting techniques.

## Technologies
- Python
- Google Colab
- Data Cleaning
- Algorithm Analysis
- Big O Notation

## Results
The experimental results showed that Heap Sort achieved the most stable performance, while Quick Sort performance depended on pivot selection, and Insertion Sort became inefficient for large datasets.
